# Machine Learning Decision Record (MLDR)

このドキュメントでは、志ん生音声再現モデルの進化（学習）の過程を記録します。

## MLDR-0001: 第1回ファインチューニング学習計画 (Phase 1: 基礎固め)

### 1. ステータス
**Proposed (提案中)**

### 2. 背景と目的
現在は学習回数0回のベースモデル（XTTS-v2 / RVC）を使用しています。第1回学習の目的は、志ん生特有の「音響的特徴（声質）」と「基礎的な落語のリズム」をモデルに定着させることにあります。

### 3. 学習戦略
- **対象モデル**: XTTS-v2 (GPT LoRA) + RVC v2
- **データセット**: 代表的な演目（例：『火焔太鼓』）から抽出した高品質な10分間
- **学習アプローチ**: 低学習率での長時間学習により、ベースモデルの汎用性を維持しつつ志ん生の質感を上書きする。

### 4. 評価指標 (Success Criteria)
1. **音響的類似度**: 志ん生本人の音声とのコサイン類似度が 0.85 を超えること。
2. **落語的リズム**: 生成された音声の「無音区間（間）」の統計的分布が、オリジナルの音源と近似していること。
3. **アーティファクト**: RVC変換後の金属音や不自然なノイズが、ベースモデルより低減されていること。
